# Changelog

## 0.3.1 — 2026-07-15

### 版本更新

- 新增“关于”页面，通过公开 GitHub Release 检测新版本，仅提醒并由用户前往下载，不执行自动更新
- 设置概览显示新版本提醒，支持直接忽略当前版本
- 支持在应用内展开并渲染 Release Log

### 智能头像构图

- 引入本地 MediaPipe 人脸检测、关键点与头发分割模型，不上传头像图片
- 头像编辑器支持智能构图，可选择脸部或头部居中、调整构图范围，并按需保护完整头部
- 设置页使用真实模型预览构图效果，缓存样张检测结果，并采用编辑、保存、取消流程
- 新增“构图全部头像”，提供进度、停止、完成汇总及统一任务日志
- 演员单次和批量刮削可选择自动智能构图；每位演员头像保存后立即构图，完成后再继续下一位
- 批量智能构图与影片、演员批量刮削强制互斥，避免并发改写头像

### 头像资源

- 保存头像原图、裁剪参数与源图指纹，重新构图不再依赖低清晰度展示图
- 改进旧头像迁移及 JPEG、PNG、GIF、WebP、AVIF 格式识别，修复扩展名与实际格式不一致导致的批量失败
- 提高头像导出清晰度，并在源图不变时保留已有手动裁剪

### 图片预览

- 修复 macOS 下图片预览窗口闪现后无法打开的问题
- 保留系统后退、鼠标返回键和 macOS 返回手势关闭预览的交互

### 文档与打包

- 补充版本规范、发布流程、图片预览历史栈及智能头像构图设计文档
- 打包本地模型与许可证文件，并在生产构建中校验模型运行资源完整性

## 0.3.0 — 2026-07-11

### 演员头像

- 头像源图与裁剪参数持久化（schema v2：`avatar_source_path` / `avatar_crop_json`）
- 头像资源路径按演员 ID 区分，避免同名演员共用文件
- 刮削下载与现有源图一致时保留手动裁剪，不覆盖
- 头像编辑器支持封面/写真取源，并改进加载与错误反馈

### 路由与导航

- 侧栏跨一级列表时恢复离开前的筛选；再点当前列表不再清空筛选
- 从影片详情进入厂牌/系列等分类时，不再带上媒体库筛选参数

### 界面

- 设置工作区、批量任务控件与数字步进器
- 统一空状态高度：固定面板铺满，详情样张/写真用中等带高
- 详情操作栏、列表表面与选择工具条等组件整理

### 修复

- 将 `strtok3` 声明为直接依赖，稳定影片时长探测

### 文档

- 补充主导航记忆、空状态高度与 UI 组件约定

## 0.2.0 — 2026-07-05

预发布版本（非正式 1.0）。

### 媒体库

- 影片扫描、番号解析、多文件关联（`video_files`）
- 海报墙虚拟滚动、多维筛选与分类浏览
- 播放清单
- 样张导入（本地文件 / 图片链接）
- 详情页元数据展示、维护信息、外部评分

### 刮削

- 插件化刮削体系（内置 JavDB、JavLibrary、JAV8 及演员插件）
- 影片默认刮削插件：JavDB；演员默认：Xslist
- 批量刮削队列与字段级更新模式
- 刮削浏览器会话、挑战页处理、UA 配置
- 用户自定义插件与插件开发助手（含 MCP）

### 演员

- 演员库、别名归并、写真导入
- 演员刮削与详情页

### 设置与网络

- 媒体库路径、资产存储迁移
- HTTP/HTTPS 代理与连接测试
- LLM 多提供商配置与元数据翻译

### 数据

- SQLite schema version 1（无增量迁移；旧开发库需删除后重建）
- 设置持久化于 `settings.json`

### 其他

- 调用系统默认播放器，无内置播放器
- `media://` 本地资产协议

### 修复与优化

- 清除元数据时一并删除外部站点评分
- 导入弹窗：空状态占满剩余高度、不预留滚动条槽位
- 写真空状态高度与样张对齐
