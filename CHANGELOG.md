# Changelog

## 0.3.0 — 2026-07-11

### 演员头像

- 头像源图与裁剪参数持久化（schema v2：`avatar_source_path` / `avatar_crop_json`）
- 头像资源路径按演员 ID 区分，避免同名演员共用文件
- 刮削下载与现有源图一致时保留手动裁剪，不覆盖
- 头像编辑器支持封面/写真取源，并改进加载与错误反馈

### 路由与导航

- 侧栏跨一级列表时恢复离开前的筛选；再点当前列表不再清空筛选
- 从影片详情进入厂牌/系列等分类时，不再带上媒体库筛选参数

### 界面

- 设置工作区、批量任务控件与数字步进器
- 统一空状态高度：固定面板铺满，详情样张/写真用中等带高
- 详情操作栏、列表表面与选择工具条等组件整理

### 修复

- 将 `strtok3` 声明为直接依赖，稳定影片时长探测

### 文档

- 补充主导航记忆、空状态高度与 UI 组件约定

## 0.2.1 — 2026-07-05

### 体验

- 未刮削引导：设置概览通知、媒体库/演员列表维护 Banner、详情页提示

### 数据同步

- 批量刮削进度全局同步至列表与概览统计
- 优化路由切换时的 refetch 策略，减少侧栏切换多余请求

### 刮削

- Cloudflare 挑战页检测更准确，减少 JavDB 正常页误判
- 挑战页工具栏新增「保存页面」

### 修复

- 筛选弹层内下拉（刮削状态、年份等）点击无效
- 无可用 LLM 提供商时默认模型选项置空

### 文档

- README 重写，补充功能说明与界面截图

## 0.2.0 — 2026-07-05

预发布版本（非正式 1.0）。

### 媒体库

- 影片扫描、番号解析、多文件关联（`video_files`）
- 海报墙虚拟滚动、多维筛选与分类浏览
- 播放清单
- 样张导入（本地文件 / 图片链接）
- 详情页元数据展示、维护信息、外部评分

### 刮削

- 插件化刮削体系（内置 JavDB、JavLibrary、JAV8 及演员插件）
- 影片默认刮削插件：JavDB；演员默认：Xslist
- 批量刮削队列与字段级更新模式
- 刮削浏览器会话、挑战页处理、UA 配置
- 用户自定义插件与插件开发助手（含 MCP）

### 演员

- 演员库、别名归并、写真导入
- 演员刮削与详情页

### 设置与网络

- 媒体库路径、资产存储迁移
- HTTP/HTTPS 代理与连接测试
- LLM 多提供商配置与元数据翻译

### 数据

- SQLite schema version 1（无增量迁移；旧开发库需删除后重建）
- 设置持久化于 `settings.json`

### 其他

- 调用系统默认播放器，无内置播放器
- `media://` 本地资产协议

### 修复与优化

- 清除元数据时一并删除外部站点评分
- 导入弹窗：空状态占满剩余高度、不预留滚动条槽位
- 写真空状态高度与样张对齐
