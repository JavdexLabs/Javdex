# Changelog

## 0.2.0 — 2026-07-05

- 影片默认刮削插件改为 JavDB
- 清除元数据时一并删除外部站点评分
- 导入弹窗：空状态占满剩余高度、不预留滚动条槽位
- 写真空状态高度与样张对齐

## 1.0.0 — 2026-07-05

首个正式版本。

### 媒体库

- 影片扫描、番号解析、多文件关联（`video_files`）
- 海报墙虚拟滚动、多维筛选与分类浏览
- 播放清单
- 样张导入（本地文件 / 图片链接）
- 详情页元数据展示、维护信息、外部评分

### 刮削

- 插件化刮削体系（内置 JavDB、JavLibrary、JAV8 及演员插件）
- 批量刮削队列与字段级更新模式
- 刮削浏览器会话、挑战页处理、UA 配置
- 用户自定义插件与插件开发助手（含 MCP）

### 演员

- 演员库、别名归并、写真导入
- 演员刮削与详情页

### 设置与网络

- 媒体库路径、资产存储迁移
- HTTP/HTTPS 代理与连接测试
- LLM 多提供商配置与元数据翻译

### 数据

- SQLite schema version 1（无增量迁移；旧开发库需删除后重建）
- 设置持久化于 `settings.json`

### 其他

- 调用系统默认播放器，无内置播放器
- `media://` 本地资产协议
