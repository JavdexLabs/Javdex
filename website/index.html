THE WORKSPACE

复杂的资料，
保持清晰。

为大型本地媒体库设计。高密度信息始终在视线内，日常维护不被多余界面打断。

01 / LIBRARY

浏览，不必翻找。

搜索、筛选、排序与多种浏览维度聚合在同一工作区，让数千条记录仍然易于定位。

02 / DETAIL

每项资料，都有来处。

封面、样张、演员、评分、标签与本地文件集中呈现，也可以随时手动修正。

03 / PEOPLE

从作品，连接到人物。

维护头像、写真、别名与关联作品，让媒体库不再只是一组互不相关的文件。

CAPABILITIES

从扫描开始，
在本地完成。

01

目录扫描

递归识别本地视频与复杂文件名，快速建立索引。

02

插件刮削

按字段组合插件，获取元数据与图片资产。

03

批量任务

可暂停、恢复或取消，并自由选择更新策略。

04

人物资料

管理头像、写真、别名以及关联作品。

05

本地资产

SQLite 与图片留在设备中，支持图片加密。

06

开放扩展

内置插件开发助手，并提供可选 MCP 服务。

LOCAL, BY DEFAULT

没有云端账户。
没有媒体上传。

Javdex 不提供任何媒体内容，也不内置在线播放服务。数据库、图片和设置默认保存在你的设备上，播放交给系统默认播放器。
