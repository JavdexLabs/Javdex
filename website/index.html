当前安装包尚未进行商业代码签名。Windows 可能显示“未知发布者”；macOS 可在“系统设置 → 隐私与安全性”中确认打开。请只从本项目 GitHub Release 下载。

02 / INTERFACE

为大型媒体库设计的工作区

高密度但不嘈杂，让筛选、检查和维护都保持在视线内。

03 / CAPABILITIES

从文件夹到可维护的资料库

围绕本地文件建立索引，元数据和图片资产始终由你掌控。

01

扫描与识别

递归扫描本地目录，识别常见视频格式和复杂文件名中的番号。

02

插件化刮削

按字段组合影片与演员插件，获取封面、样张、写真和结构化资料。

03

批量任务

暂停、恢复或取消批量刮削，并选择补齐、覆盖或重匹配策略。

04

演员资料

维护演员头像、写真、别名与关联影片，并支持本地头像构图。

05

本地资产

SQLite 数据库与图片资产保存在设备中，支持图片加密存储。

06

可扩展工作流

内置插件开发助手，并提供可选 MCP 服务连接外部开发工具。

LOCAL FIRST

你的文件、数据库和图片，
都在你的设备里。

Javdex 不提供任何媒体内容，也不内置在线播放服务。播放时调用系统默认播放器， 本地人脸检测同样无需上传图片。
